# Geoloc-fe
A Front-End geolocation project in React Native. The app allows you to geolocate on a map using Google Maps, a tab allows you to search for addresses in France via the government API and to add GPS coordinates to a personalized list. This list is found on the Map tab and displays position markers at each of the addresses on the map.
